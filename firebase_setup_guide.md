# คู่มือการตั้งค่า Google Firebase สำหรับระบบกล่องแสดงความคิดเห็น

เนื่องจากหน้าเว็บของเราจัดเก็บอยู่บน GitHub Pages (ไม่มีเครื่องแม่ข่ายหรือฐานข้อมูลส่วนตัว) เราจึงต้องใช้ระบบฐานข้อมูลฟรีของ Google ที่มีชื่อว่า **Google Firebase** เพื่อทำหน้าที่เสมือน "สมุดบันทึก" ที่เก็บข้อความที่คนอื่นพิมพ์มาให้กำลังใจเราครับ

คุณครูสามารถทำตามขั้นตอนการสมัครและตั้งค่าแบบทีละขั้นตอน (Step-by-Step) ด้านล่างนี้ได้เลยครับ:

---

## ขั้นตอนที่ 1: ลงทะเบียนและสร้างโปรเจกต์บน Firebase
1. เปิดบราวเซอร์แล้วเข้าไปที่เว็บไซต์: [https://console.firebase.google.com/](https://console.firebase.google.com/)
2. เข้าสู่ระบบด้วยบัญชี **Gmail** ของคุณครู
3. คลิกที่ปุ่ม **"Create a project"** (สร้างโปรเจกต์)
4. ตั้งชื่อโปรเจกต์ เช่น `practice-library-comments` จากนั้นกดปุ่ม **Continue**
5. ในหน้าการเปิดใช้งาน Google Analytics ให้เลื่อนปิดสวิตช์เป็น **Disabled** (เพื่อความรวดเร็วและง่ายในการตั้งค่า) แล้วกดปุ่ม **Create project**
6. รอระบบเตรียมพื้นที่สักครู่ จากนั้นกด **Continue** เพื่อเข้าสู่หน้าจัดการหลัก

---

## ขั้นตอนที่ 2: สร้างฐานข้อมูลข้อความ (Realtime Database)
1. ที่เมนูด้านซ้ายมือ ให้คลิกคำว่า **Build** จากนั้นเลือกหัวข้อ **Realtime Database**
2. คลิกปุ่ม **"Create Database"** (สร้างฐานข้อมูล)
3. ระบบจะให้เลือกตำแหน่งเซิร์ฟเวอร์ ให้คงค่าเริ่มต้นไว้ (เช่น สหรัฐอเมริกา หรือสิงคโปร์) แล้วกด **Next**
4. ระบบจะให้เลือกโหมดเริ่มต้น ให้เลือก **"Start in test mode"** (เริ่มในโหมดทดสอบ) เพื่อให้ทุกคนสามารถส่งข้อความได้โดยไม่ต้องล็อกอิน จากนั้นกดปุ่ม **Enable**
5. *ยินดีด้วยครับ! ตอนนี้คุณครูมีฐานข้อมูลออนไลน์สำหรับเก็บบันทึกข้อความแล้ว*

---

## ขั้นตอนที่ 3: เปิดใช้งานการเชื่อมต่อหน้าเว็บ (Register App)
1. คลิกที่รูปเฟืองฟันเฟือง (ตั้งค่า) ตรงเมนูด้านซ้ายบน ข้างคำว่า **Project Overview** แล้วเลือก **Project settings** (การตั้งค่าโปรเจกต์)
2. เลื่อนลงมาด้านล่างสุดในหัวข้อ **Your apps** แล้วคลิกที่ไอคอนรูปวงเล็บ **`</>`** (Web App)
3. ตั้งชื่อแอปพลิเคชันของคุณครู เช่น `practice-library-web` (ไม่จำเป็นต้องติ๊กเลือก Firebase Hosting) แล้วคลิกปุ่ม **Register app**
4. ระบบจะแสดงรหัสการตั้งค่า (Firebase Configuration) ขึ้นมา หน้าตาจะคล้ายๆ แบบนี้ครับ:

```javascript
const firebaseConfig = {
  apiKey: "AIzaSy...",
  authDomain: "practice-library-comments.firebaseapp.com",
  databaseURL: "https://practice-library-comments-default-rtdb.firebaseio.com",
  projectId: "practice-library-comments",
  storageBucket: "practice-library-comments.appspot.com",
  messagingSenderId: "123456789...",
  appId: "1:123456789..."
};
```

5. ให้คัดลอกส่วนที่เป็นรหัสภายในปีกกา `{ ... }` เหล่านี้เก็บไว้

---

## ขั้นตอนที่ 4: นำรหัสมาเชื่อมต่อกับหน้าเว็บของคุณครู
1. เปิดไฟล์ `index.html` ของโปรเจกต์
2. ค้นหาแถวที่มีคำว่า **`const firebaseConfig = {`** (จะอยู่ประมาณบรรทัดที่ 330)
3. นำรหัสที่คุณครูคัดลอกมาจากขั้นตอนที่ 3 มาวางแทนที่ค่าเดิม ดังนี้:
   - นำค่า `apiKey` มาใส่แทน `"YOUR_API_KEY"`
   - นำค่า `authDomain` มาใส่แทน `"YOUR_PROJECT_ID.firebaseapp.com"`
   - นำค่า `databaseURL` มาใส่แทน `"https://YOUR_PROJECT_ID-default-rtdb.firebaseio.com"`
   - นำค่า `projectId` มาใส่แทน `"YOUR_PROJECT_ID"`
   - นำค่า `storageBucket` มาใส่แทน `"YOUR_PROJECT_ID.appspot.com"`
   - นำค่า `messagingSenderId` มาใส่แทน `"YOUR_MESSAGING_SENDER_ID"`
   - นำค่า `appId` มาใส่แทน `"YOUR_APP_ID"`
4. บันทึกไฟล์ `index.html` แล้วอัปเดตไฟล์ขึ้น GitHub 

---

## การตรวจสอบ
เมื่อคุณครูทำการวางรหัสเสร็จสิ้น หน้าเว็บจะเปลี่ยนจากป้ายแจ้งเตือนเป็น **กล่องพิมพ์ข้อความคอมเมนต์** ทันที และเมื่อมีนักเรียนหรือคนอื่น ๆ พิมพ์ความคิดเห็นเข้ามา ข้อความเหล่านั้นจะถูกจัดเก็บในระบบ Firebase และนำมาแสดงผลเรียงลำดับเวลาให้ทุกคนได้เห็นทันทีครับ!
